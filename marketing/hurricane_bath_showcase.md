# Hurricane Bath - website showcase (drop-in content)

**What this is.** The finished content for the Hurricane Bath showcase, the hero of Clean's
bath-forward marketing. It is written to drop into the Astro marketing site when that is
scaffolded. No component, page, or animation is built yet, on purpose (per `elons_algorithm`,
the build waits for the site; what is scarce now is the content, not the code).

**Why it matters.** The Hurricane Bath is Clean's un-promptable differentiator, so this is a
`dig_the_moat` asset: it sells what the bath does and that Paul invented it, while keeping the
build private.

**Status: DRAFT.** Real claims, photos, and numbers come from Paul. Anything not yet confirmed
is marked as a data gap. Nothing here is invented (`real_data_only`); marketing copy is exactly
where fabricated specs creep in, so this doc keeps drafts grounded and flags every gap.

## The moat decision (read first)

**Reveal the what and the story, protect the how.** The public page sells the outcome (a faster,
deeper, calmer bath that gets through coats a standard rig cannot) and the fact that Paul built
it himself over years. It does NOT publish the build recipe (the pump configuration, the valve,
the plumbing). That stays in the private record (CLEAN_FIELD_MANUAL.md, "The Hurricane Bath") so
a competitor cannot copy it off the site.

Recommended public framing: "a custom dual-pump system Paul built to move far more water than
standard grooming rigs," shown with real proof, and no schematic. Confirm this is the line you
want before the copy is finished.
