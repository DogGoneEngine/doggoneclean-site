# Brand assets

Canonical, cleaned-up Dog Gone Clean logos. Grab them from here rather than
re-exporting.

| File | Notes |
|---|---|
| `dog-gone-clean-logo.png` | 910x345, transparent background (RGBA) |

This is the transparent version of the site logo. The solid white background was
cut away so the mark sits cleanly on any color (a dark panel, a colored card, a
photo) with no white box around it. The white dog body is preserved.

The white-background original still lives at `public/logo.png`, which is what the
marketing site nav and the Laelaps admin use on their light surfaces, where a
white background is invisible anyway. Reach for this transparent copy whenever a
logo needs to sit on something that is not white.
